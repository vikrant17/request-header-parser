
# Request Header Parser

      
    
  
## User stories:
   
    
* I can get the IP address, language and operating system for my browser.
   



  
## Example usage:
 `https://request-header-parser17.glitch.me/whoami`
    
    
    
## Example output:
   
  
     `{
        "ipaddress": "107.293.128.188",
        "language": "en-US",
        "software": "Windows NT 10.0; Win64; x64"
      }`
